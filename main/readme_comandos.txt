                            COMANDOS DO README.md

- Títulos são dados por # => h1, ## => h2 e assim por diante
    - # => h1
    - ## => h2
    - ### => h3
    - #### => h4
    - ##### => h5
    - ###### => h6

- Textos normais funcionam sem nenhum código apenas escrevendo
    - mas podemos dar ênfase no texto com comandos 
    - negrito **texto** ou __texto__
    - itálico *texto* ou __texto__
    - itálico e negrito _um **texto**combinado_

- Listas ordenadas e não ordenadas
    - ordenadas numeos com ponto na frente 1.  2.  3.
        1. um
        2. dois
        3. três
    - não ordenadas se usa colocando * na frente do texto
        * arroz
        * feijão
        * batata

- Colocar imagens coloca a sintaxe ![Texto alt](link imagem ou pasta img);
    - ![uma floresta](link ou pastas img/uma floresta.png)
    - a imagem pode estar no repo ou em link externo

- Colocar link e parecido com o de imagem mas a sintaxe é [Texto do link](link)
    - [Google](https://www.google.com/)
    - e se for link do github pode ser direto apenas o link mesmo

- Colocar código no readme e feito por ```código```
    - ``` function soma(a + b){
            return a + b;
        }
      ```
    - exclusivo do markdown do github

- Fazer lista de tarefas 
    - -[x] => para tarefas completadas
    - -[ ] => para tarefas nao completadas
    - -[x] conclui o css da home
    - -[ ] não conclui o css da home
